# Employee Management System

Full stack CRUD application using:

Frontend: React + Bootstrap  
Backend: Spring Boot  
Database: MySQL  

## Features
- Add employee
- View employees
- Update employee
- Delete employee

## Tech Stack
- React
- Spring Boot
- REST APIs
- MySQL
- Axios
- Bootstrap

## How to Run

### Backend
Open ems-backend in IntelliJ or Eclipse  
Run:
EmsBackendApplication.java  

Backend runs on:
http://localhost:8080

### Frontend
cd ems-frontend
npm install
npm run dev

Frontend runs on:
http://localhost:4029
